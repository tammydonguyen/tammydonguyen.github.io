For more basic problems, see the topic-based guides and practice problems!</p> --- <a name="wwpp" class="anchor"></a> ## What would Python print? from operator import sub this = lambda a, b : lambda c: a(b, c) d = 30 def that(e, f): if e and f: e, d = f(sub, e)(5), e else: print(e) d = 20 def this(g): return lambda : print(g(d, e)) return this The code above is loaded into the Python interpreter. What would the following lines output? Assume they are entered one after the other in the same environment. If there is no output, leave the space blank.
